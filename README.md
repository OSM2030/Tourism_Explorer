## Project Overview  
**Tourism Explorer** is a Python + Flask web application that helps users explore nearby tourist locations by entering a place name or geographical coordinates.  
The system displays results on a Folium interactive map and generates a Matplotlib visitor-count chart.

This project demonstrates:
- Web development using Flask  
- Data manipulation using Pandas  
- Geocoding using Geopy  
- Data visualization using Matplotlib  
- Interactive mapping using Folium   
- Proper use of Python virtual environments & best practices  

---

## Python Libraries Used  
This project uses **5 major Python libraries**, satisfying the minimum requirement:
Flask==2.3.2
pandas==2.2.2
folium==0.14.0
matplotlib==3.8.2
geopy==2.4.0

Each library has a role:

| Library | Purpose |
|--------|---------|
| **Flask** | Creates web server, routes, and templates |
| **Pandas** | Loads and processes spots dataset (CSV) |
| **Folium** | Generates interactive maps with markers |
| **Matplotlib** | Creates visitor count bar charts |
| **Geopy** | Geocodes place names + calculates distances |
